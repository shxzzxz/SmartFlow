# 各场景流程示例

本文以具体的业务场景为例，展示系统数据表的操作情况。

**前置假定：系统中已存在以下基础账户数据：**

* `1` (支付宝，ASSET)
* `2` (招商银行，ASSET)
* `3` (信用卡，LIABILITY)
* `4` (公司报销，ASSET - 应收款)
* `5` (基金，ASSET)
* `101` (餐饮，EXPENSE)
* `201` (工资，INCOME)
* `301` (理财盈亏，EQUITY)

## 场景 A：日常消费支出

* **场景类别**：单笔支出
* **场景描述**：使用支付宝购买一杯咖啡，花费 20 元。
* **场景复式记账说明**：餐饮支出增加（借方），支付宝资产减少（贷方）。
* **具体表记录操作情况**：

`transactions` 表插入：

```json
{
  "id": 1,
  "tx_type": "EXPENSE",
  "amount": 20.00,
  "description": "买咖啡",
  "tx_time": "2023-10-24 10:00:00"
}
```

`entries` 表插入：

```json
[
  { "id": 1, "transaction_id": 1, "account_id": 101, "flow_direction": "DEBIT", "amount": 20.00 },
  { "id": 2, "transaction_id": 1, "account_id": 1, "flow_direction": "CREDIT", "amount": 20.00 }
]
```

## 场景 B：工资收入确认

* **场景类别**：单笔收入
* **场景描述**：收到本月工资 10,000 元，打入招商银行卡。
* **场景复式记账说明**：招商银行资产增加（借方），工资收入增加（贷方）。
* **具体表记录操作情况**：

`transactions` 表插入：

```json
{
  "id": 2,
  "tx_type": "INCOME",
  "amount": 10000.00,
  "description": "10月工资",
  "tx_time": "2023-10-25 09:00:00"
}
```

`entries` 表插入：

```json
[
  { "id": 3, "transaction_id": 2, "account_id": 2, "flow_direction": "DEBIT", "amount": 10000.00 },
  { "id": 4, "transaction_id": 2, "account_id": 201, "flow_direction": "CREDIT", "amount": 10000.00 }
]
```

## 场景 C：信用卡还款

* **场景类别**：资产与负债流转（内部转账）
* **场景描述**：从招商银行卡转出 5,000 元用于还信用卡。此操作不涉及收支统计。
* **场景复式记账说明**：信用卡负债减少（借方），招商银行资产减少（贷方）。
* **具体表记录操作情况**：

`transactions` 表插入：

```json
{
  "id": 3,
  "tx_type": "TRANSFER",
  "amount": 5000.00,
  "description": "还信用卡",
  "tx_time": "2023-10-26 12:00:00"
}
```

`entries` 表插入：

```json
[
  { "id": 5, "transaction_id": 3, "account_id": 3, "flow_direction": "DEBIT", "amount": 5000.00 },
  { "id": 6, "transaction_id": 3, "account_id": 2, "flow_direction": "CREDIT", "amount": 5000.00 }
]
```

## 场景 D：公费垫付与报销核销

* **场景类别**：垫款与应收账款流转
* **场景描述**：先用个人信用卡垫付公司差旅机票 2,000 元；次周公司财务将 2,000 元打入个人招商银行卡。该过程不产生个人实际收支。
* **场景复式记账说明**：
  * **动作 1（垫付）**：报销账户(应收款)资产增加（借方），信用卡负债增加（贷方）。
  * **动作 2（收款核销）**：招商银行资产增加（借方），报销账户(应收款)资产减少（贷方）。
* **具体表记录操作情况**：

**动作 1（垫付记录）：**

`transactions` 表插入：

```json
{
  "id": 4,
  "tx_type": "TRANSFER",
  "amount": 2000.00,
  "description": "垫付机票"
}
```

`entries` 表插入：

```json
[
  { "id": 7, "transaction_id": 4, "account_id": 4, "flow_direction": "DEBIT", "amount": 2000.00 },
  { "id": 8, "transaction_id": 4, "account_id": 3, "flow_direction": "CREDIT", "amount": 2000.00 }
]
```

**动作 2（收款核销记录）：**

`transactions` 表插入：

```json
{
  "id": 5,
  "tx_type": "TRANSFER",
  "amount": 2000.00,
  "description": "公司报销款到账"
}
```

`entries` 表插入：

```json
[
  { "id": 9, "transaction_id": 5, "account_id": 2, "flow_direction": "DEBIT", "amount": 2000.00 },
  { "id": 10, "transaction_id": 5, "account_id": 4, "flow_direction": "CREDIT", "amount": 2000.00 }
]
```

## 场景 E：理财盈亏与账面平账

* **场景类别**：系统平账（无现金流变动）
* **场景描述**：记录持有的基金账面亏损 500 元。
* **场景复式记账说明**：所有者权益（盈亏账户）减少（借方），基金资产减少（贷方）。
* **具体表记录操作情况**：

`transactions` 表插入：

```json
{
  "id": 6,
  "tx_type": "ADJUSTMENT",
  "amount": 500.00,
  "description": "基金亏损平账"
}
```

`entries` 表插入：

```json
[
  { "id": 11, "transaction_id": 6, "account_id": 301, "flow_direction": "DEBIT", "amount": 500.00 },
  { "id": 12, "transaction_id": 6, "account_id": 5, "flow_direction": "CREDIT", "amount": 500.00 }
]
```
