# 用户体验细节

## 1. 快速记账流程

**最少操作步骤：**
1. 点击”记一笔”按钮。
2. 输入金额（自动聚焦）。
3. 选择分类（常用分类置顶）。
4. 点击”完成”。


**智能默认值：**
* 账户：默认选择对应分类上一次的账户。
* 时间：默认当前时间，支持快速调整。
* 分类：根据历史记录推荐。


## 2. 视觉反馈

**金额显示：**
* 支出：红色，前缀 “-”。
* 收入：绿色，前缀 “+”。
* 转账：灰色，前缀 “→”。


**账户余额变化：**
* 记账后显示账户余额变化动画（如：¥1000 → ¥980）。


## 3. 错误处理

**数据校验：**
* 金额不能为 0 或负数。
* 必填字段：金额、账户、分类。
* 转账时，转出账户和转入账户不能相同。


**友好提示：**
* “该交易可能重复，是否继续？”
* “预算已超支 ¥200，是否继续记账？”
