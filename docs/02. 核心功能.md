# 核心功能

本文档描述 SmartFlow 的功能蓝图，重点说明系统需要支持哪些能力

## 模块 1：现金流管理

**数据录入：**

- **手动记账：** 支持普通支出、普通收入、转账、退款、报销、还款、借入、期初余额与余额调整。
- **已入账交易更正：** 已入账记录不直接覆盖，通过“冲销原交易 + 重记正确交易”完成金额、账户、分类、备注或时间修正。
- **已入账交易删除：** 删除等价于冲销且不补记，不物理删除已入账记录。
- **账单导入：** 支持一木记账、支付宝、微信、国内主流银行的 CSV / Excel 账单导入，流程包含字段映射、预览、调整与提交。
- **周期性账单：** 支持用户预设周期规则，到期自动生成交易。

**流水展示：**

- 按时间倒序展示交易流水。
- 支持按时间、账户、分类、业务用途和关键字筛选。
- 退款、报销到账等子记录默认折叠到父交易详情中。
- 详情页可查看更正、冲销与关联子记录链路。

**预算控制：**

- 支持总预算和分类预算。
- 提供超支预警和预算趋势。
- 支持整笔交易不计入预算。
- 分类迁移时保留预算归属。

**数据分析：**

- **结构分析：** 展示收入 / 支出分类占比。
- **趋势分析：** 展示月度、半年、一年等周期的收支变化。
- **对比分析：** 支持同环比（MoM / YoY）等基础指标。

## 模块 2：资产负债管理

管理用户的资产、负债与净资产视图。底层使用统一账户表表达资产、负债、权益、收入与支出。

**资产账户：**

- **资金账户：** 现金、储蓄卡、第三方支付等。
- **理财账户：** 基金、股票等，支持通过余额调整记录浮亏浮盈。
- **报销应收账户：** 用于记录“公司应付我”的应收款，报销到账时核销。

**负债账户：**

- **信贷账户：** 信用卡、花呗、白条等，支持账单日、还款日和额度信息。
- **债务账户：** 房贷、车贷、个人借款等。

**账户视图与操作：**

- 创建、编辑、隐藏、排序、设置图标与备注。
- 展示资产、负债、净资产汇总。
- 支持账户流水查看和账户间转账。

## 模块 3：分类管理

收入 / 支出分类与资产 / 负债账户共享统一账户模型，页面上仍以分类树方式管理。

- 独立维护收入分类树和支出分类树。
- 支持新增、编辑、隐藏、排序和设置图标。
- 支持一级 / 二级分类。
- 分类删除时根据历史引用决定物理删除或归档迁移。
- 分类预算和历史交易在迁移后保持可追溯。

## 模块 4：智能化规则与自动化

为了降低记账与导入整理成本，系统提供自动化能力。

**混合分类引擎：**

- **规则引擎：** 用户可配置关键词或正则规则，优先进行本地匹配。
- **AI 辅助分类：** 导入账单无规则命中时，可使用模型根据交易对手、备注和历史分类推断分类。

**信贷账户辅助：**

- 待还金额计算。
- 账单日与还款日提醒。
- 信贷账户账单周期聚合视图。

**周期性账单：**

- 支持固定周期支出、收入或转账。
- 自动生成的交易与手动交易遵循同一更正和删除规则。

## 模块 5：偏好与通用设置

**财务周期设置：**

- 账单月起始日：默认为 1 号，支持自定义
- 日历周起始日：默认为周日，支持自定义

**显示设置：**

- 分类精简显示。
- 流水页是否显示具体时间。
- 退款等场景是否显示原金额。
- 流水列表是否显示账户图标。

**数据导出与备份：**

- 支持 CSV / Excel 导出。
- 支持自定义导出字段和时间范围。
- 支持完整本地数据库备份与恢复。

## 数据安全与隐私

- 所有数据默认存储在设备本地
- 不上传财务数据到服务器，除非用户主动启用同步或 AI 辅助能力。
- 备份与恢复由用户主动触发。
- 后续可扩展 WebDAV、端到端加密云盘等同步通道。

## 未来扩展

以下方向不作为 v1 核心目标：

- 多账本、家庭账本与多人协作。
- 投资行情同步、收益率计算和资产配置分析。
- 银行 API、支付宝 / 微信授权自动同步。
- 异常消费检测和更完整的智能提醒。
- iOS 平台支持。
